# Arrays

## Problems Covered:
- Linear Search (Easy)
- Largest Number in Array (Easy)
- Binary Search (Easy)
- Reverse Array (Easy)
- Maximum Subarray Sum (Bruteforce technique) (Easy)
- Maximum Subarray Sum (Kadane's Algorithm) (Medium)
- Trapping Rainwater (Hard)
- Best Time to Buy and Sell Stock (Easy)
- Prefix Sum (Medium)

## Key Learnings:
- Optimizing brute force to O(n)
- Using prefix sums for faster calculations
- Understanding sliding window concepts
